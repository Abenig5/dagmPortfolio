# Personal Portfolio Website

Modern responsive portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- Responsive Design
- Dark/Light Mode
- Smooth Animations
- SEO Optimized
- Fast Performance
- Contact Form
- Project Showcase
- Skills Section
- Experience Timeline

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
- Vercel Deployment
